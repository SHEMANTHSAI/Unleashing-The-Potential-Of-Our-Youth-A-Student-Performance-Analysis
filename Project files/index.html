About Us

Exploratory Data Analysis (EDA) is an approach for data analysis and data exploration that employs a variety of techniques (mostly graphical representation) on the data we are working on.

  • Univariate Analysis
  • Bivariate Analysis
  • Multivariate Analysis

In this dataset, we have many features like Grades, Average, Sum, Parental level of Education, Test Preparation course, individual Subject scores like Math score, Reading score, Writing Score, Gender, discount, and the Dependent variable is Result. The remaining are independent variables.

Exploratory Data Analysis (EDA) is an approach that is used to analyze the data and discover trends, patterns, or check assumptions in data with the help of statistical summaries and graphical representations.
